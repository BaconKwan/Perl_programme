数据库注释与靶向预测

数据库注释

鉴定到的环状RNA通过局部比对软件 blast 比对到该物种的 circBase 数据库中。以E值 < e-10 作为过滤条件，找出高度可信的被注释到的环状RNA并定义为已存在的环状RNA，反之为新预测的环状RNA。

对于被 circBase 数据库收录的环状RNA， StarBase 数据库提供了这些环状RNA与小RNA的靶向关系。因此，通过 StarBase 数据库可以找到存在的环状RNA与小RNA的靶向关系。

环状RNA注释信息表说明：

  • Gene ID (gene_id) ：环状RNA ID
  • Annotation (annotation) ：该环状RNA在circBase中收录的类型
  • Best Transcript (best transcript) ：该环状RNA来源基因的NCBI编号
  • Gene Symbol (gene symbo) ：该环状RNA来源基因的名称
  • Study (circRNA study) ：报道该环状RNA的研究
靶向关系预测

对于所有的环状RNA与该物种的全体小RNA还可能存在一些未经报道的靶向关系。所以，分析还会进行所有环状RNA和该物种的小RNA进行靶向关系预测。动物样品用mireap，miranda，targetscan三个软件进行靶向关系预测，植物样品用patmatch软件来进行靶向关系预测。得到的结果为新预测的靶向关系。

环状RNA已存在(新预测)靶向关系信息表说明：

  • gene_id ：环状RNA ID
  • source_gene ：该环状RNA的来源基因，“NA”表示没有来源基因，说明该环状RNA类型为“intergenic”
  • chr ：该环状RNA的来源染色体
  • strand ：正负链
  • genomic_start ：该环状RNA在基因组上的起始位点
  • genomic_end ：该环状RNA在基因组上的终止位点
  • spliced_length ：该环状RNA成环后的长度
  • annot_type ：该环状RNA所属的类型
  • miRNA ：该环状RNA的存在靶向关系的小RNA列表
