样品（分组）表达量统计表说明

  • Gene ID (id)：环状RNA ID
  • XXX RPKM (xxx_rpkm)：样品XXX该环状RNA的表达量
  • XXX Count (xxx_count)：样品XXX比对到该环状RNA的read数目
  • Source Gene (source_gene) ：该环状RNA的来源基因，“NA”表示没有来源基因，说明该环状RNA类型为“intergenic”
  • Chr (chr) ：该环状RNA的来源染色体
  • Strand (strand) ：正负链
  • Start (genomic_start) ：该环状RNA在基因组上的起始位点
  • End (genomic_end) ：该环状RNA在基因组上的终止位点
  • Length (spliced_length) ：该环状RNA成环后的长度
  • Type (annot_type) ：该环状RNA所属的类型
样品(分组)间差异表达分析

差异基因注释表

  • id ：环状RNA ID
  • sample_1_count ：样品1 的环状RNA count 值
  • sample_2_count ：样品2 的环状RNA count 值
  • sample_1_rpkm ：样品1 的环状RNA RPKM 值
  • sample_2_rpkm ：样品2 的环状RNA RPKM 值
  • log2(FC) ：样品1 与 样品2 RPKM 差异倍数的对数值，以2为底
  • P_value ：检验统计量的 P-value 值
  • FDR ：经过FDR校正后的 P-value 值
  • significant ：值为yes或no，yes表示为显著差异，判定条件：若 FDR < 0.05，并且 |log2FC| > 1 此值就为 “yes”
  • source_gene ：该环状RNA的来源基因，“NA”表示没有来源基因，说明该环状RNA类型为“intergenic”
  • chr ：该环状RNA的来源染色体
  • strand ：正负链
  • genomic_start ：该环状RNA在基因组上的起始位点
  • genomic_end ：该环状RNA在基因组上的终止位点
  • spliced_length ：该环状RNA成环后的长度
  • annot_type ：该环状RNA所属的类型

样品(分组)间差异表达模式聚类分析

我们基于基因表达量，对样本和基因间的关系进行层级聚类，并使用热图来呈现聚类结果。我们对各个样本的基因表达量以 2 为底求对数值，对不同样品和基因进行层级聚类分析，图中每列代表一个样品，每行代表一个基因，基因在不同样品中的表达量用不同颜色表示，颜色越红表示表达量越高，颜色越蓝表示表达量越低。

样品间差异表达模式聚类分析，选取的是样品间显著差异基因并集；分组间差异表达模式聚类分析，选取的是分组间显著差异基因并集。
