表达量统计

在计算环状RNA表达量的时候，会遇到环状RNA和线性RNA有相同的Exon区域，而Reads无法准确分配的情况。这时为了计算的准确性，我们只取环状RNA的Support Reads，也就是跨越Break Point的Reads，而丢弃掉其他Reads。

因此，环状RNA表达量的计算使用RPM(Reads Per Million mapped reads)法，设RPM(A)为基因A的表达量，则C为比对到基因A的测序片段数，N为比对到参考基因的总测序片段数，其计算公式为：

样品(分组)间差异表达分析

差异基因注释表

  • id : 转录本名称
  • sample_1_count : 样品1 的转录本 count 值
  • sample_2_count : 样品2 的转录本 count 值
  • sample_1_rpm : 样品1 的转录本 RPM 值
  • sample_2_rpm : 样品2 的转录本 RPM 值
  • log2(FC) : 样品1 与 样品2 RPM 差异倍数的对数值，以2为底
  • P_value : 检验统计量的 P-value 值
  • FDR : 经过FDR校正后的 P-value 值
  • significant : 值为yes或no，yes表示为显著差异，判定条件：若 FDR < 0.05，并且 |log2FC| > 1 此值就为 “yes”
  • gene_id : 该转录本所属的基因名称
  • source_gene : 该转录本所属的基因名称
  • chr : 转录本所在的染色体
  • strand : 转录本所在的单链
  • genomic_start : 转录本在基因组上的起始位置
  • genomic_end : 转录本在基因组上的终止位置
  • spliced_length : 转录本剪切加工，成环后的长度
  • annot_type : 该转录本的类型

样品(分组)间差异表达模式聚类分析

我们基于基因表达量，对样本和基因间的关系进行层级聚类，并使用热图来呈现聚类结果。我们对各个样本的基因表达量以 2 为底求对数值，对不同样品和基因进行层级聚类分析，图中每列代表一个样品，每行代表一个基因，基因在不同样品中的表达量用不同颜色表示，颜色越红表示表达量越高，颜色越蓝表示表达量越低。
