环状RNA鉴定

Anchors Reads比对到参考基因组上后，将所有样品的比对结果进行合并。得到合并后的结果后使用 find_circ 进行环状RNA鉴定。对于环状RNA鉴定结果，我们进行过滤得到高度可信的环状RNA。过滤条件如下：

  1. breakpoints = 1，只保留有且只有1个清晰breakpoint的环状RNA
  2. anchor_overlap <= 2，每条Reads的两个Anchors Reads比对到基因组上的位置overlap不能超过2bp
  3. edit <= 2，只允许2bp错配
  4. n_uniq > 2，uniq的Reads大于2条
  5. best_qual_A > 35 或 best_qual_B > 35，每条Reads的其中一个Anchors Reads比对到基因组上最好的Mapping结果要比其排第二的结果的分值高35分以上
  6. n_reads > samples，支持该环状RNA的reads要大于支持它的样品数
  7. 环状RNA的长度小于100k

环状RNA信息统计

环状RNA信息统计表说明：

  • chrom ：染色体
  • start ：基因组上的起始位置
  • end ：基因组上的终止位置
  • name ：环状RNA ID
  • n_reads ：支持该环状RNA的Reads数
  • strand ：正负链
  • n_uniq ：支持该环状RNA的Uniq Reads数
  • best_qual_A ：Anchors_Reads_A最好的Mapping结果的分值
  • best_qual_B ：Anchors_Reads_B最好的Mapping结果的分值
  • tissues ：支持该环状RNA的样品
  • tiss_counts ：支持该环状RNA的每个样品的reads数
  • edits ：碱基编辑位点数
  • anchor_overlap ：Anchors Reads的overlap位点数
  • breakpoints ：该环状RNA的断点数
  • score ：断点两端在基因组上的距离
环状RNA类型统计

结合 find_circ 给出的环状RNA信息和参考基因组的转录组注释，可对鉴定到的环状RNA进行类型判定。

主要类型：

  • A - annot_exon ：环状RNA的breakpoint位于已知基因的一个外显子的起始和另一个外显子的终止，其序列由breakpoint之间的外显子的碱基构成
  • A - one_exon ：环状RNA位于已知基因的某一个外显子内，其序列由breakpoint之间的所有碱基构成
  • B - exon_intron ：环状RNA的breakpoint之间有已知基因的一个或多个完整外显子但breakpoint不位于已知外显子的起始或终止，其序列由上breakpoint起始位点到第一个外显子起始位点、breakpoint之间的外显子和最后一个外显子到breakpoint终止位点的碱基构成
  • B - intronic ：环状RNA位于已知基因的某一个内含子内，其序列由breakpoint之间的所有碱基构成
  • C - antisense ：环状RNA位于已知基因的反义链，其序列由反义链上breakpoint之间的所有碱基构成
  • D - intergenic ：环状RNA位于已知基因的之间，其序列由breakpoint之间的所有碱基构成

环状RNA类型统计表说明：

  • Gene ID (gene_id) ：环状RNA ID
  • Source Gene (source_gene) ：该环状RNA的来源基因，“NA”表示没有来源基因，说明该环状RNA类型为“intergenic”
  • Chr (chr) ：该环状RNA的来源染色体
  • Strand (strand) ：正负链
  • Start (genomic_start) ：该环状RNA在基因组上的起始位点
  • End (genomic_end) ：该环状RNA在基因组上的终止位点
  • Length (spliced_length) ：该环状RNA成环后的长度
  • Type (annot_type) ：该环状RNA所属的类型
